# gerenciador de Questões com Spring MVC
Webapp para gerenciar as questões dos meus apps de simulados de concursos engenharia.

## Tecnologias utilizadas
1- Java

2- Spring MVC

3- Jpa

4- Hibernate

5- MySQL

6- Thymeleaf
